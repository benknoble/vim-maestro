# Files

> samples/code/{a,b}.txt

## Buffers

- in-memory version of file
- `:ls`
- `:bn`, `:bp`, `:b`
- `:w`
- `:e`
- `hidden`
- `:wall` and `:qall`

## Windows

- view-port onto buffer
- `:sp`, `:vs`
- `<C-w>`

## Tabs

- Not like tabs in other editors!
- Collection of windows
- `:tabedit`, `gt`
